## Default Permission

Allows CloudCAD subscribers to access the CloudCAD API

- `allow-get-subscriber`

## Permission Table 

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

</td>
<td>

</td>
</tr>

<tr>
<td>

`subscriber-client:deny-get-subscriber`

</td>
